DEPLOYMENT WORKFLOW
================================================================================

STEP 1: GitHub Repository Setup
  [ ] Create private repo: torgor-and-tweed
  [ ] Push code: git push -u origin main
  [ ] Add GitHub Secrets:
      - VPS_HOST (IP or domain)
      - VPS_USER (SSH username, usually 'torgor')
      - DEPLOY_PRIVATE_KEY (SSH private key)
      - DOMAIN (production domain)

STEP 2: VPS Provisioning (Automated)
  [ ] SSH to Hostinger VPS: ssh root@YOUR_VPS_IP
  [ ] Download script: scp scripts/deploy.sh root@IP:/tmp/
  [ ] Run script: bash /tmp/deploy.sh
  [ ] Verify: node -v, npm -v, pm2 -v, nginx -v

STEP 3: Manual Setup on VPS
  [ ] SSH as torgor: ssh torgor@YOUR_VPS_IP
  [ ] Clone repo: git clone https://github.com/USER/torgor-and-tweed.git /home/torgor/app
  [ ] Create .env.local (copy from .env.example, fill secrets)
  [ ] npm ci
  [ ] npx prisma migrate deploy
  [ ] npm run build

STEP 4: Start Application
  [ ] pm2 start ecosystem.config.js
  [ ] pm2 save
  [ ] pm2 startup
  [ ] Verify: curl http://localhost:3000

STEP 5: SSL Certificate (Let's Encrypt)
  [ ] Point domain DNS to VPS IP
  [ ] Run: sudo certbot certonly --nginx -d your-domain.com
  [ ] Update /etc/nginx/sites-available/torgor-and-tweed with SSL paths
  [ ] sudo nginx -t && sudo systemctl reload nginx
  [ ] Verify: curl -I https://your-domain.com

STEP 6: Post-Deployment
  [ ] Test homepage, products, search, checkout
  [ ] Verify security headers (curl -I https://domain)
  [ ] Check console for errors
  [ ] Monitor: pm2 logs torgor-and-tweed
  [ ] Submit sitemap to Google Search Console

TROUBLESHOOTING QUICK REFERENCE
================================================================================

App won't start:
  pm2 logs torgor-and-tweed
  curl http://localhost:3000

Database connection error:
  cat .env.local | grep DATABASE_URL
  mysql -h HOST -u USER -p DATABASE -e "SELECT 1;"

SSL certificate issues:
  sudo certbot certificates
  sudo certbot renew --dry-run

Nginx errors:
  sudo nginx -t
  sudo systemctl restart nginx
  sudo tail -f /var/log/nginx/error.log

High memory usage:
  pm2 monit
  pm2 logs torgor-and-tweed --err
  pm2 restart torgor-and-tweed
